## 1. 过拟合是什么？怎么解决？

**答：**
过拟合指模型在训练集表现很好但测试集差（泛化能力弱）。

**原因**：数据太少/模型太复杂/训练太久

**解决**：1) 增加数据/数据增强 2) 正则化(L1/L2/Dropout) 3) 早停 4) 减少模型复杂度 5) 集成学习

---

## 2. L1和L2正则化的区别？

**答：**
- **L1(Lasso)**：λΣ|w|，产生稀疏解（部分权重=0），可做特征选择
- **L2(Ridge)**：λΣw²，权重趋近0但不为0，一般不解出0

**为什么L1产生稀疏**：L1在0点不可导，优化更容易把权重推向0。

---

## 3. 梯度消失/爆炸是什么？怎么解决？

**答：**
链式求导时，梯度连乘小于1的数→消失；连乘大于1的数→爆炸。

**解决**：1) ReLU/GELU激活 2) BatchNorm 3) 残差连接 4) 好的初始化(Xavier/Kaiming) 5) 梯度裁剪(防爆炸)

---

## 4. 为什么Transformer要用Self-Attention？

**答：**
Self-Attention = softmax(QK^T/√d_k)V

**优点**：1) **捕捉长距离依赖** 2) **并行计算**（比RNN快） 3) 每个位置直接交互，信息损失少

除以√d_k原因：防止维度大时内积过大导致softmax饱和，梯度消失。

## 5. RLHF的三个阶段是什么？

**答：**

| 阶段 | 描述 |

## 6. 为什么需要Batch Normalization？

**答：**
BN对batch的每个特征做标准化：(x-μ_B)/σ_B，再用γ和β恢复表达能力。

**作用**：1) 加速收敛（允许大学习率） 2) 缓解梯度消失 3) 轻微正则化效果 4) 降低对初始化敏感度

**训练用batch统计量，推理用滑动平均统计量**。

---

## 7. Dropout的原理和作用？

**答：**
训练时以概率p随机丢弃神经元。推理时乘以(1-p)保持期望一致。

**作用**：1) 减少神经元间的共适应 2) 隐式集成多个子网络 3) 防过拟合

## 8. 如何解决类别不平衡问题？

**答：**
1) **过采样/欠采样**（SMOTE合成少数类样本）
2) **类别加权**：class_weight='balanced'
3) **Focal Loss**：聚焦难分类样本
4) **评估用F1/AUC**，不用Accuracy

## 9. 传统机器学习模型的区别（常考选择）

**答：**
- **LSTM**适合文本生成等序列任务
- **HMM**适合序列标注
- **决策树**适合分类/回归，可解释性好
- **Random Forest**=Bagging+决策树，降低方差
- **GBDT/XGBoost**=Boosting+决策树，降低偏差

## 10. 偏差方差分解

**答：**
泛化误差 = 偏差² + 方差 + 噪声

- 偏差大→欠拟合
- 方差大→过拟合
- 需要在两者间权衡

## 13. 手写LSTM公式

```python
# 拼接输入和隐状态
combined = concat(x_t, h_{t-1})
# 四个门
f = sigmoid(W_f @ combined + b_f)  # 遗忘门
i = sigmoid(W_i @ combined + b_i)  # 输入门
o = sigmoid(W_o @ combined + b_o)  # 输出门
c_tilde = tanh(W_c @ combined + b_c) # 候选状态
# 更新
c_t = f * c_{t-1} + i * c_tilde
h_t = o * tanh(c_t)
```

## 14. 手写逻辑回归训练步骤

```python
# 1. 标准化
X_norm = (X - mean) / std
# 2. 加偏置
X_norm = np.hstack([np.ones((N,1)), X_norm])
# 3. 初始化
w = np.zeros(D+1)
# 4. 梯度下降循环
for _ in range(epochs):
    pred = sigmoid(X_norm @ w)  # 前向
    grad = (1/N) * X_norm.T @ (pred - y)  # 求导
    w -= lr * grad  # 更新
```

## 16. 什么是MoE？门控网络的作用？

**答：**
MoE = 多个专家(FFN) + 门控网络

**门控网络**：为每个token计算到各专家的分数，选Top-k个专家处理

**关键点**：门控网络参数也会训练（不冻结★），负载均衡损失防坍缩

## 17. FlashAttention的原理？

**答：**
利用SRAM（片上高速缓存），通过分块tiling+重计算，减少HBM（显存）访问次数。

**内存复杂度从O(n²)降到O(n)**，且与标准Attention数值等价。

---

## 18. 模型量化？

**答：**
把FP32参数用INT4/INT8表示。

INT4：32bit→4bit，存缩小约87.5%，推理速度大幅提升。

方法：PTQ（训练后量化，如GPTQ、SmoothQuant）和QAT（量化感知训练，如QLoRA）。

## 19. RAG的原理？

**答：**
RAG = 检索增强生成。先检索外部知识库→把检索结果拼接进prompt→让大模型基于检索内容生成。最直接缓解大模型幻觉的方法。

## 20. 为什么要用位置编码？

**答：**
Self-Attention本身**没有顺序信息**（词袋模型），需要位置编码让模型知道词序。

常见方案：正余弦编码、可学习位置编码、RoPE旋转位置编码。

RoPE可通过基频缩放微调后外推更长上下文。
